# Gestion des réservations d'événements

Application web développée en PHP permettant de gérer les réservations d'événements de manière simple et efficace.

## 🚀 Fonctionnalités
- **CRUD complet :** Créer, lire, modifier et supprimer des réservations.
- **Interface intuitive :** Navigation fluide pour les utilisateurs.
- **Base de données :** Gestion sécurisée des données avec SQL.

## 🛠️ Stack Technique
- **Backend :** PHP, SQL (PDO)
- **Frontend :** HTML, CSS
- **Outils :** Git, VS Code

## ⚙️ Installation
1. Clonez ce dépôt :
   `git clone https://github.com/Thehussssam/Gestion-des-r-servations-d-v-nements.git`
2. Importez le fichier SQL dans votre base de données (via phpMyAdmin).
3. Configurez votre connexion à la base de données dans le fichier de configuration.
4. Lancez le projet via un serveur local (XAMPP/WAMP).
